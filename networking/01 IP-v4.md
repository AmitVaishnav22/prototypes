# IPv4 Basics

## What is an IP Address?
- Unique identifier for every device on a network.
- IP = Internet Protocol.
- Localhost = `127.0.0.1`.

## IPv4 Structure
- IPv4 = 32-bit address.
- Written in **dotted decimal** format.
- Consists of **4 octets**.

Example:
192.168.14.1

## Octet
- 1 octet = 8 bits.
- Range: 0–255.

Example:
192 | 168 | 14 | 1

## Why 32 Bits?
- 4 octets × 8 bits = **32 bits**
- 32 bits = **4 bytes**

## Binary Representation
Example:

192.168.14.1

↓

11000000.10101000.00001110.00000001

## Important Terms
- IPv4
- Octet
- Dotted Decimal Notation
- Binary Representation
- Localhost

## Remember
- IP addresses are stored in **binary**.
- Humans read them in **decimal**.
- Binary is used for subnetting, routing, and networking calculations.
